# Repo Risk Intelligence

> AI-powered Engineering Risk Intelligence for GitHub repositories.

## Overview

Repo Risk Intelligence (BusFactor.fyi) helps engineering teams identify knowledge concentration, contributor dependency, and repository risk before they become operational problems.

> **Current Status:** MVP (Work in Progress)

### Current MVP

- Repository risk assessment
- Bus factor scoring
- Contributor ownership analysis
- Executive report generation
- Knowledge graph visualization
- Live web application

## Live Demo

https://bus-factor-risk-anal-1g9y.bolt.host

## Why it exists

Engineering teams often accumulate hidden knowledge silos. This project visualizes contributor concentration and repository risk so teams can make better engineering decisions.
