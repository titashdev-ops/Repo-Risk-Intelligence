# ARCHITECTURE

> Product: **Repo Risk Intelligence (BusFactor.fyi)**

## Purpose

Repo Risk Intelligence helps engineering teams identify knowledge concentration, contributor dependency, and repository risk using repository metadata and contribution patterns.

---

# High-Level Flow

```text
GitHub Repository
        │
        ▼
Repository Analysis
        │
        ▼
Risk Assessment Engine
        │
        ├── Bus Factor
        ├── Ownership Distribution
        ├── Contributor Analysis
        │
        ▼
Knowledge Graph
        │
        ▼
Executive Report
```
